# RL Robot Logistician

Projet de Reinforcement Learning (Q-Learning vs DQN) pour un robot autonome capable de livrer des colis dans un entrepôt dynamique tout en minimisant la consommation d'énergie et en évitant les obstacles.

## Objectifs
- Implémenter Q-Learning tabulaire sur une grille simple.
- Développer un Deep Q-Network (DQN) avec Replay Buffer et Target Network.
- Comparer les performances et analyser la convergence des politiques.

## Structure du projet
- `environment/` : environnement de simulation
- `notebooks/` : notebooks d’entraînement
- `results/` : courbes et métriques
- `demo/` : vidéo de démonstration
- `report/` : rapport comparatif
